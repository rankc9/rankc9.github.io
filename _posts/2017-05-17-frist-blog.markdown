#### What's this

[rankc.cc](http://rankc.cc) 是由rankc在GitHub上创建的一个新的博客页面,主题搬运自[Vno Jekyll](https://github.com/onevcat/vno-jekyll).

#### Usage

```bash
$ git clone https://github.com/onevcat/vno-jekyll.git your_site
$ cd your_site
$ bundler install
$ bundler exec jekyll serve
```
